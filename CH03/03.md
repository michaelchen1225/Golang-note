## if

### 最簡單的 if 敘述

```go
if <布林運算式> {
    // 程式區塊
}
```

範例：檢查奇偶數

```go
package main

import (
	"fmt"
	"log"
)

var x int

func main() {

	fmt.Print("請輸入一個整數：")
	_, error := fmt.Scanf("%d", &x)

	if error != nil {
		fmt.Println("輸入錯誤")
		log.Fatal(error)
	}

	if x%2 == 0 {
		fmt.Println("偶數")
	}

	if x%2 == 1 {
		fmt.Println("奇數")
	}
}
```

### if、else、else if

改良上述範例，使用 if-else & else if 讓程式更簡潔：

```go
package main

import (
	"fmt"
	"log"
)

var x int

func main() {

	fmt.Print("請輸入一個整數：")
	_, error := fmt.Scanf("%d", &x)

	if error != nil {
		fmt.Println("輸入錯誤")
		log.Fatal(error)
	} else if x%2 == 0 {
		fmt.Println("偶數")
	} else {
		fmt.Println("奇數")
	}
}
```

### if 的起始賦值

有些變數只在 if 中會用到，那就在 if 開頭中宣告就好，把變數範圍限制在 if 中：

```go
if <起使賦值>; <布林運算式> {
    // 程式區塊
}
```

「起始賦值」只能適用於下列情況：

* 短變數宣告，例如：`x := 1`、`x := funcA()`

* 算數 or 邏輯運算式，例如：`x + 1`、`(x + 1) == 2`

* 遞增 or 遞減，例如：`x++`、`x--`

範例：輸入數字比需大於 7 並小於 20，再判斷奇偶數

```go
package main

import (
    "fmt"
    "errors"
    "log"
)

var x int

func inputAndCheck(input int) error {

    if input < 7 || input > 20 {
        return errors.New("輸入錯誤，請輸入 7 ~ 20 之間的整數")
    } else {
        return nil
    }
}

func main() {
    fmt.Print("請輸入一個整數：")
    _, error := fmt.Scanf("%d", &x)

    if error != nil {
        fmt.Println("輸入錯誤")
        log.Fatal(error)
    } else if error := inputAndCheck(x); error != nil {
        fmt.Println(error)
    } else if x%2 == 0 {
        fmt.Println("偶數")
    } else {
        fmt.Println("奇數")
    }
}
```

## switch

### 基本語法

```go
switch <起始賦值>; <運算式> {
	case <運算式>:
		// 程式區塊
	case <運算式>, <運算式>:
		// 程式區塊
		fallthrough
    default:
		// 程式區塊
}
```

* 開頭的「起始賦值」與「運算式」都可以自由挑選，都不寫就是「switch true」。

* 「switch true」的用法有點像 if-else：哪個 case 為 true 就執行哪個 case。

* 如果一個 case 符合條件，執行該 case 後就會跳出 switch。如果要繼續比對下一個 case，可以使用 `fallthrough`。

* 如果沒有符合的 case，就會執行 default。(可選)

### 範例：判斷星期幾

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	x := time.Friday

	switch x {
	case time.Monday:
		fmt.Println("星期一")
	case time.Tuesday:
		fmt.Println("星期二")
	case time.Wednesday:
		fmt.Println("星期三")
	case time.Thursday:
		fmt.Println("星期四")
	case time.Friday:
		fmt.Println("星期五")
	case time.Saturday:
		fmt.Println("星期六")
	case time.Sunday:
		fmt.Println("星期日")	
	default:
		fmt.Println("錯誤")
	}
}
```
```
星期五
```

### 範例：檢查今天是否周末(星期六、日)：

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	switch x := time.Now().Weekday(); x {
	case time.Saturday, time.Sunday:
		fmt.Println("周末", x)
	default:
		fmt.Println("平日", x)
	}
}
```
```
平日 Wednesday
```

另一種寫法：

```go
package main

import (
	"fmt"
	"time"
)

func main() {
   
	switch x:= time.Now().Weekday(); {
		case x == time.Saturday || x == time.Sunday:
			fmt.Println("周末", x)
		default:
			fmt.Println("平日", x)
	}
}
```
```
平日 Wednesday
```

## 迴圈 (for)

Golang 只有 `for` 一種迴圈，但可以達成 while 的效果。

### 基本語法

```go
for <起始賦值>; <條件>; <結束敘述> {
	// 程式區塊
}
```
* 起始賦值：同 if 的起始賦值。

* 條件：迴圈的條件，每次迴圈檢查是否為 true。

* 結束敘述：「每次」迴圈結束後執行的敘述，通常用來遞增或遞減。

* 起史賦值、條件、結束敘述皆可以自由組合，如果都省略等同於「for true」，會無限迴圈。

### 範例：印出 1 ~ 5

```go
package main

import "fmt"

func main() {

	for i :=1; i <=5; i++ {
		fmt.Println(i)
	}

}
```
```
1
2
3
4
5
```

### 模擬 while 迴圈

```go
for <條件敘述> {
	// 程式區塊
}
```

範例：計算 1 到 10 的總和

```go
package main

import "fmt"

func main() {

	sum := 0
	i := 1

	for i <= 10 {
		sum += i
		i++
	}

	fmt.Println(sum)
}
```
```
55
```

### 使用 for i 走訪陣列

```go
package main

import "fmt"

func main() {

	nums := []int{1, 2, 3, 4, 5}

	for i := 0; i < len(nums); i++ {
		fmt.Print(nums[i], " ")
	}

}
```
```
1 2 3 4 5 
```

### 使用 for range -- 走訪陣列、切片、map、字串

`for range` 可以用來走訪陣列、切片、map 等等：

```go
for <index>, <value> := range <array or slice or map or string> {
	// 程式區塊
}
```

如果只需要 value，可以用 `_` 來省略 index：

```go
for _, <value> := range <array or slice or map> {
	// 程式區塊
}
```

反之，如果只需要 index，可以省略 value：

```go
for <index> := range <array or slice or map> {
	// 程式區塊
}
```

範例：走訪陣列

```go
package main

import "fmt"

func main() {

	nums := []int{1, 2, 3, 4, 5}

	for i, v := range nums {
		fmt.Printf("index: %d, value: %d\n", i, v)
	}

}
```
```
index: 0, value: 1
index: 1, value: 2
index: 2, value: 3
index: 3, value: 4
index: 4, value: 5
```
只需要 value：

```go
package maipackage main

import "fmt"

func main() {

	nums := []int{1, 2, 3, 4, 5}

	for _, v := range nums {
		fmt.Printf("value: %d\n", v)
	}

}

```
```
value: 1
value: 2
value: 3
value: 4
value: 5
```

### 使用 for range --- 生成數列

`range n` 可以用來生成整數數列，範圍是 0 到 n-1：

```go
for i := range n {
	// 程式區塊
}
```

範例：印出 1 到 5

```go
package main

import "fmt"

func main() {

	for i := range 5 {
		fmt.Println(i + 1)
	}

}
```
```
1
2
3
4
5
```

### break 與 continue

* break 跳出迴圈。

* continue 跳過本次迴圈，馬上開始下一次迴圈。
